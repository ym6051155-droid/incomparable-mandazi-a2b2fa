حزمة موقع SUST Library الكاملة
==============================

ارفع جميع الملفات الموجودة داخل هذا المجلد إلى جذر استضافة الموقع، وليس ملف index.html وحده.

الملفات الأساسية:
- index.html: الموقع
- library-manifest.json: قائمة الملفات التي يعرضها الموقع
- manifest.webmanifest + sw.js + الأيقونات: تثبيت الموقع كتطبيق PWA والعمل عند ضعف الإنترنت
- offline.html: صفحة عدم الاتصال
- cloudflare-r2-worker.js: ربط اختياري تلقائي مع Cloudflare R2
- wrangler.toml.example: مثال إعداد ربط الـWorker بالـBucket

الميزات المضافة:
- زر تحميل واحد فقط لكل ملف
- أحدث الملفات
- المفضلة المحفوظة على الهاتف
- متابعة آخر ملف
- البحث مع مرشحات السمستر والقسم
- عرض نوع الملف وحجمه وتاريخ إضافته عند توفرها
- رسالة واضحة للأقسام الفارغة
- الإبلاغ عن رابط لا يعمل عبر واتساب
- عداد الملفات
- تثبيت الموقع على الشاشة الرئيسية
- حفظ الصفحات وملفات PDF المفتوحة للعمل عند ضعف الإنترنت

طريقة إضافة ملفات يدويًا:
أضف كل ملف داخل files في library-manifest.json.
الموقع يدعم الحقول:
courseRoot, category, number, file, labelAr, labelEn, size, addedAt, url

الربط التلقائي مع R2:
1. أنشئ Cloudflare Worker.
2. الصق كود cloudflare-r2-worker.js.
3. أضف R2 Binding باسم SUST_LIBRARY واربطه بالـBucket.
4. اجعل مسار الـWorker هو /api/library على نفس نطاق الموقع، أو ضع رابطه داخل:
   window.SUST_LIBRARY_API = 'رابط الـWorker/api/library';
   قبل سكربت الموقع.
5. عندها سيقرأ الموقع الملفات من R2 تلقائيًا، ثم يستخدم library-manifest.json كنسخة احتياطية.
