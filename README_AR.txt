تم تجهيز الأيقونة المرفوعة مع:
- وضوح أعلى وحواف أكثر حدة.
- لمعان خفيف واحترافي.
- خلفية وإطار أبيضان.
- المقاسات المطلوبة لأندرويد.
- Adaptive Icon للأجهزة الحديثة.
