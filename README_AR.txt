مكتبة SUST — مركز التنبيهات V14

الملفات الجديدة:
- index.html: الموقع مع جرس التنبيهات.
- notifications.json: قاعدة رسائل التنبيهات.
- admin-notifications.html: صفحة إعداد الرسائل.

النشر:
1. ارفع كل ملفات الحزمة إلى جذر مستودع GitHub المرتبط بـNetlify.
2. استبدل الملفات القديمة.
3. اضغط Commit changes.
4. افتح الموقع:
   https://sust-library.site/?v=14

إرسال رسالة جديدة:
1. افتح:
   https://sust-library.site/admin-notifications.html
2. اكتب الرسالة واضغط «إضافة الرسالة».
3. اضغط «تنزيل notifications.json».
4. ارفع notifications.json إلى جذر مستودع الموقع في GitHub واستبدل القديم.
5. اضغط Commit changes.
6. بعد نجاح Netlify يظهر التنبيه في الجرس.

ملاحظة:
هذه تنبيهات داخل الموقع. تظهر عند فتح الموقع أو أثناء بقائه مفتوحًا.
لا تظهر على شاشة الهاتف والموقع مغلق.
